# CLI Applications

Recipes for building command-line applications in Python.

Command-line applications are one of the simplest and most productive ways to create useful tools.

Many personal utilities, automation scripts, developer tools, and AI applications can be implemented as CLIs.

This cookbook contains recipes for building modern Python command-line applications.
